# Quickstart Examples

Get started with DAO in under 5 minutes. These examples demonstrate the fundamental concepts with minimal code.

## Examples

| File | Description | Time |
|------|-------------|------|
| [01_basic_usage.py](01_basic_usage.py) | Create your first DataStore and DataObject | 2 min |
| [02_multiple_interfaces.py](02_multiple_interfaces.py) | Using multiple interfaces per DataStore | 3 min |

## Prerequisites

- DAO library installed (`uv sync`)
- No external services required (uses mock interfaces)

## Running Examples

```bash
# From the repository root
python examples/quickstart/01_basic_usage.py
python examples/quickstart/02_multiple_interfaces.py
```
